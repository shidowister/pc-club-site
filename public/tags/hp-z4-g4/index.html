【ジャンク再生計画 Vol.4】「ECCメモリ」って何？ サーバー用メモリの安さと罠

CPU、GPUときて、今回は 「メモリ」 です。 もともとこのZ4G4にはmicron製のDDR4の8GBのメモリが２枚挿さっていました。 普通のパソコンなら「DDR4の8GBをもう何枚か買えばOK！」で終わる話ですが、相手はワークステーションでCPUはXeon。そう簡単にはいきません。 今回は、我々を悩ませる 「ECCメモリ」 と、その種類のについて解説します。 1. そもそも「ECCメモリ」とは？ 普通のPC用メモリ（Non-ECC）と違い、ECC（Error Correction Code） メモリには「データのエラーを自動で訂正する機能」がついています。 普通: なんらかのエラーでデータが1ビット化けた → ブルースクリーンで落ちる ECC: データが化けた → 自動で直して動き続ける 「24時間365日止まってはいけない」サーバーやワークステーションのための機能です。 この機能のおかげで、Xeonマシンは非常に安定しているわけですね。 このECC機能の影響で、動作が1クロック分遅くなるという弱点はありますが、大体2000Mhzを超える動作の1クロックは誤差の範囲です。 2. 「Registered」と「Unbuffered」の罠 ここからが本題です。 実はXeonで使えるECCメモリには、大きく分けて2種類あります。 ECC UDIMM (Unbuffered): 普通のメモリにECC機能がついただけのもの。一部のCore iシリーズや一部のRyzenでも使えたりする。 ECC RDIMM (Registered): 信号を整える「レジスタ」チップが載ったガチ業務用。 この2つ、形は同じDDR4なのに混ぜると動きません。 そしてマザーボードによって「どっちしか対応していない」という縛りがあったりします。 今回のHP Z4 G4は、公式には 「RDIMM」 が推奨されています。（UDIMMも動く場合がありますが、混在は不可） 3. なぜサーバー用メモリは安いのか？ ヤフオクやメルカリを見ると、中古なら普通のDDR4メモリより、サーバー用（RDIMM）の方がちょっと安いことに気づきます。 「8GBで1000円！？」みたいなことがたまにあります。 理由は単純。 「データセンターの世代交代で大量に廃棄されるから」 そして 「普通のPCには刺さらないから」 です。 需要と供給の関係で、我々のようなジャンクPC愛好家にとって、RDIMMは 「宝の山」 なのです。 もちろん業務用なので、普通に買うとびっくりするぐらい高いです。 4. 今回の戦利品 というわけで、今回確保したメモリがこちら。 Sk hynix製 DDR4-2666V ECC RDIMM 8GB × 3枚 = 24GB ...

前回紹介したベースマシン「HP Z4 G4」。 今回は、ゲーミングPCの頭脳となる CPUの換装 について、その選定プロセスを記録します。 「ソケットが合うから何でも載る」と思ったら大間違い。そこにはワークステーション特有の深い沼がありました。 1. マザーボードの仕様と「C422」の制約 まず、このマシンのソケットは LGA2066 です。 自作PCに詳しい方なら、「お、Core i9-10900Xとか、Core i7-9800Xといった『Core Xシリーズ』が載るやつじゃん！」と思うかもしれません。 私も最初はそう思いました。 しかし、Z4 G4のチップセットは 「Intel C422」 です。 Intel X299チップセット: コンシューマ向け。Core i9 / i7 Xシリーズに対応。 Intel C422チップセット: 業務用。Xeon W-2000シリーズ専用。 そう、物理的にソケットの形状が同じでも、Core iシリーズは認識しないのです（排他仕様）。 この時点で、高性能な中古のCore i9を載せる夢は断たれ、「Xeon W-2000シリーズ」 の中から選ぶことになりました。 ジーク・Xeon! 2. Xeon W-2000シリーズの型番を解読する XeonのcpuはジオンのMS同様に種類が多くてわかりにくいので、今回のターゲットとなる「W-2000番台」の命名規則を整理しました。 例： Xeon W-2235 W: Workstation向け（シングルソケット用） 2: シリーズ区分（メインストリーム） 2: 世代（1=Skylake-W, 2=Cascade Lake-W） 35: 性能ランク（数字が大きいほどコア数が増える） Z4 G4の初期搭載CPUは「W-2102」だったので、Skylake-Wのエントリーモデルでした。 今回は、脆弱性対策が進み、クロックあたりの性能も微増している Cascade Lake-W / W-22xx のものを中心に探すことにしました。 3. 候補CPU比較：ゲーミングに最適なのは誰だ？ ゲーム用途で最も重要なのは「コア数」よりも 「クロック周波数」 です（厳密にはシングル性能）。 いくらコアが多くても、1コアあたりの速度が遅いとFPSが伸びません。 ...

【ジャンク再生計画 Vol.1】予算11万円でPS5 proに勝つ！現状編

部員S藤が某所から貰ってきた5年落ちのワークステーション。 もともと業務用のミドルクラス機なので電源や拡張性など、ポテンシャルはあります。 このマシンのパーツを換装して、低価格でハイスペpcを目指します！ 今回のコンセプト ただ組むだけじゃ面白くありません。今回のテーマはps5 pro以下の費用で、ps5 pro以上のスペック、汎用性を目指すことです。 現状のpcの紹介 1. 本体 HP社の業務用ワークステーション Z4G4。 法人向けとしてはエントリーミドルクラスで、発売当時の新品価格は構成にもよりますが、30万から40万ほどです。 ワークステーションのためcpuはxeonシリーズを使用し、gpuもquadroシリーズ、メモリもECCメモリを採用しています。 いわゆるミドルタワーで、電源や内部スペースには余裕があります。 2. cpu: [xeon w-2102] Z4G4で選べる最小構成のcpu。2017年発売。 14nmプロセスの、4コア 4スレッドのモデル。性能は高いとは言えませんが、xeonシリーズだけあって最大メモリは512GBまで対応しています。 性能的にはi5-7400より少し低いぐらい。　最近のゲームをプレイするのはちょっと厳しいです。 3. gpu: [quadro p620] ワークステーション用のquadroシリーズ。 TDPは40W。VRAMは2GB、ロープロ対応で、４系統のMini DisplayPortで4画面同時出力ができます。 ゲーミング性能はGTX950より少し低いぐらい。 最近のゲームには向きませんが、通常のビデオカードとしてみればロープロで発熱も少なく優秀と言えます。 4. 電源 HPの専用品。750Wで多分80 plus gold。 一応ATX規格らしいですが、他のマザボで使えるのかは不明です。 グラボ用の電源ピンは6ピンが二つあります。 次回、換装パーツの選定編。そもそもどんなパーツが使えるのか、どれぐらいアップデートできるのか!?
